Curso de HTML5 com CSS3

A linguagem HTML5 atualmente em sua 5° versão é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet. É a quinta versão da linguagem HTML. Esta nova versão traz consigo importantes mudanças quanto ao papel do HTML no mundo da Web, através de novas funcionalidades como semântica e acessibilidade. CSS3 é a terceira mais nova versão das famosas CSS, pela qual se define estilos para um projeto web. Com efeitos de transição, imagem, imagem de fundo/"background" e outros, pode-se criar estilos únicos para seus projetos web, alterando diversos aspectos de design no layout da página.

Curso de Javascript

JavaScript (às vezes abreviado para JS) é uma linguagem leve, interpretada e baseada em objetos com funções de primeira classe, mais conhecida como a linguagem de script para páginas Web. Os códigos em JS são executados dentro do navegador e dão acesso aos elementos HTML por meio do DOM.
